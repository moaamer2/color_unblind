# Android/iOS App

This folder contains the Flutter project for the mobile version of the Color Blindness Aid app.

## Setup

1. Make sure Flutter is installed.
2. Configure Supabase credentials if needed.
3. Run the app:
   - Android: `flutter run` or open in Android Studio
   - iOS: `flutter run` or open in Xcode

## Notes

- Requires connection to trained models for full functionality.
- The app includes simulation, real-time correction, video/image correction, and generation features.
